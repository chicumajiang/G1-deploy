# 机器人控制映射

> 适用于仿真环境（MuJoCo）和真实机器人

---

## 启动命令

```bash
# 仿真
python deploy_mujoco/deploy_mujoco.py

# 真实机器人
python deploy_real/deploy_real.py
```

## 基本控制

| 功能 | 键盘 | 手柄 |
|------|------|------|
| **LOCO模式** | `1` | `B` |
| **PASSIVE模式** | `R` | `X` |
| **位置重置** | `0` | `A` |
| **退出程序** | `Ctrl+C` | `Y` |

## 运动控制（LOCO模式）

| 功能 | 键盘 | 手柄 |
|------|------|------|
| 前进/后退 | `W` / `S` | 左摇杆 ↑↓ |
| 左移/右移 | `A` / `D` | 左摇杆 ←→ |
| 左转/右转 | `Q` / `E` | 右摇杆 ←→ |
| 重置速度 | `Space` | - |

## 手柄布局

```
        L2                    R2
        L1                    R1
    
      ↑                    Y (退出)
    ←   →          X (阻尼)   B (LOCO)
      ↓                    A (重置)
    
   左摇杆               右摇杆
```

**基本按键（单独按）：**
- `X` = PASSIVE模式
- `A` = 位置重置
- `B` = LOCO模式
- `Y` = 退出

**技能组合键：**
- `R1 + X/Y/B` = 基础技能
- `L1 + X/Y/B/A` = BeyondMimic 1-4
- `L2 + X/Y/B/A` = BeyondMimic 5-8
- `R2 + X/Y/B` = BeyondMimic 9-11

## 快速参考

### 操作流程
1. 按 `1` (键盘) 或 `B` (手柄) 进入LOCO模式
2. 使用 W/A/S/D/Q/E 或摇杆控制移动
3. 按对应按键触发技能
4. 技能完成后按 `1` 或 `B` 返回LOCO

### 紧急停止
- 键盘：`R` (PASSIVE) 或 `Ctrl+C`
- 手柄：`X` (PASSIVE) 或 `Y`
